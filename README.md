# Tanya Sharma — Portfolio

Personal portfolio website for Tanya Sharma, Data Analyst.

**Live:** [tanyasharma.vercel.app](https://tanyasharma.vercel.app)

---

## Stack

- **React 18** + **TypeScript** + **Vite**
- **Three.js** + **@react-three/fiber** — animated 3D particle network
- **GSAP** + ScrollTrigger — scroll-driven animations
- **CSS Modules** — scoped per-component styles

## Sections

| # | Section | Description |
|---|---------|-------------|
| 01 | Landing | Full-viewport 3D data-node particle network, hero text |
| 02 | About | Bio, key impact metrics, Judo athlete story |
| 03 | What I Do | Data Analytics · Product Analytics · AI/NLP · BI & Visualization |
| 04 | Work | 5 featured projects with tech stacks and metrics |
| 05 | Skills | Animated marquee across 20 technologies |
| 06 | Career | Experience timeline · Education · Certifications |
| 07 | Contact | Email, LinkedIn, GitHub |

## Run locally

```bash
npm install
npm run dev
```

## Content

All portfolio content (bio, projects, experience, skills) lives in one file:

```
src/data/index.ts
```

## Deploy

Deployed on Vercel. Push to `main` → auto-deploys.
